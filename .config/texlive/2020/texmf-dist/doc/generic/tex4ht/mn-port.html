Establishing a Port

A port can be established in the following manner.

  1. Set a directory, say,‘~/tex4ht.dir’.
  2. Download the file tex4ht.zip into the directory tex4ht.dir and unzip it.
  3. Add to ‘tex4ht.dir/texmf/tex4ht/base/’ and ‘tex4ht.dir/bin/’ a subdirectory named, say, mydir.
  4. Copy ‘tex4ht.dir/texmf/tex4ht/base/unix/tex4ht.env’ or ‘tex4ht.dir/texmf/tex4ht/base/win32/tex4ht.env’ into ‘tex4ht.dir/texmf/tex4ht/base/mydir/tex4ht.env’. Modify the file to meet the setting of your environment.
  5. Compile the source files ‘tex4ht.dir/temp/tex4ht.c’ and ‘tex4ht.dir/temp/t4ht.c’ into executable programs tex4ht and t4ht, and move the latter programs into ‘tex4ht.dir/bin/’.
  6. Set a driver for the t4ht utility.
